# Expense Tracker
Asimple and effiient Java based application to track your daily expenses and manage  your budget

# Features 
- Add new expense with categories
- Categorize spending (Food, Transportation, Shopping ,etc.)
- View monthly summarises
- Analyse spensing patterns by category
- Automatic data saving

# Prerequisites
- Java 17 or higher
- Basic command line  knowledge

# How to run code
javac src/*.java src/models/*.java src/services/*.java src/ui/*.java
java -cp src Main
